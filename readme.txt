-----------------------
apkDecompiler
ver 1.0.0
-----------------------
このアプリケーションは既存のアプリケーションの利用を簡単にしたものです。
以下のことを行います
apkファイルのapktoolによるデコンパイル
classes.dex、classes2.dex...のdex2jarによるjar化
jarを解凍、classファイルのjavaファイルへのデコンパイル

各ソフトウェアのライセンスも確認してください。

--usage--
このアプリケーションの実行には以下のものが必要です。事前に準備してください。
java実行環境
python実行環境

デコンパイルしたいapkファイルをmain.batにドラッグ＆ドロップします。
その後暫く待つとapkファイルのあったフォルダにapkファイルと同名のフォルダが作られます。
その中にデコンパイルされたソースがあります。

INFO:このアプリケーションのデコンパイルはしばしば失敗します。
